# AI Training Syllabus

**Full-Time Self-Study Program: 37.5 hours per week**
**Total Duration: 8 weeks (300 hours)**

This comprehensive training program covers Artificial Intelligence tools including Claude Code, Microsoft 365 Copilot, and foundational AI principles with a focus on UK Government security compliance (NCSC, CIS Level 2, FIPS 140-3).

## Learning Objectives Summary

By completing this syllabus, learners will be able to:

1. **Understand AI Fundamentals**: Grasp core AI concepts, machine learning basics, and responsible AI principles
2. **Master Claude Code**: Proficiently use Claude Code for software development, automation, and team collaboration
3. **Utilize M365 Copilot**: Effectively leverage Copilot across Microsoft 365 applications for productivity enhancement
4. **Apply Security Standards**: Implement NCSC Secure Design Principles, CIS Benchmark Level 2, and FIPS 140-3 requirements
5. **Engineer Effective Prompts**: Craft secure, effective prompts for various AI systems
6. **Build Secure AI Applications**: Develop AI-powered applications with appropriate security controls
7. **Ensure Compliance**: Create documentation and implement processes for regulatory compliance
8. **Demonstrate Ethical AI Usage**: Apply responsible AI principles in all work

---

## Assessment Criteria

- **Self-Assessment Questionnaire**: Pass score 80% or higher
- **Final Quiz**: Pass score 85% or higher
- **Capstone Project**: Demonstrates proficiency in all key areas with security compliance
- **Continuous Learning**: Active engagement with all resources and completion of hands-on exercises
